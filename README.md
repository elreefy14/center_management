## how to deploy 

firebase logout

# Login again
firebase login

# Set the project
firebase use future14-afdbc

# Create the target
firebase target:apply hosting release future14-afdbc-c4278

# Build the web app
flutter build web

# Deploy
firebase deploy --only hosting:release
